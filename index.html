⚙️ 科目・URL・キーの設定
ふだんは空のままでOK。共有サーバー経由で動きます。 自分のキーを使いたい場合（共有の上限を気にせず使いたい等）だけ入力してください。
▶ 自分のAPIキーの取り方（任意・5分）
  1. Google Cloud Console を開いてGoogleでログイン。
  2. 画面上部のプロジェクト選択（左上）→「新しいプロジェクト」→ 名前は何でもOK（例: study）→「作成」。
  3. 左上のメニュー（≡）→「APIとサービス」→「ライブラリ」。
  4. 検索欄に YouTube Data API v3 と入れて開く →「有効にする」。
  5. 左メニュー「APIとサービス」→「認証情報」→ 上の「＋認証情報を作成」→「APIキー」。
  6. AIza… で始まるキーが出る →「コピー」。
  7. 上の入力欄に貼り付ける（自動で保存されます）。
URL・キー・進捗・記録をまとめて1ファイルに。別のPCやブラウザに引っ越せます。
